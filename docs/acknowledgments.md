# Acknowledgments

## FastAPI and Sebastián Ramírez ([@tiangolo](https://github.com/tiangolo))

**FasterAPI** ([`faster-api-web` on PyPI](https://pypi.org/project/faster-api-web/)) takes direct inspiration from **[FastAPI](https://github.com/fastapi/fastapi)** — the API style, the focus on OpenAPI, and the idea that Python web APIs can be both ergonomic and fast.

**Sebastián Ramírez** created and leads **FastAPI**. This project exists in that lineage: a thank-you for showing what a modern Python API framework can look like, and for the ecosystem around it.

- FastAPI repository: [github.com/fastapi/fastapi](https://github.com/fastapi/fastapi)  
- FastAPI documentation: [fastapi.tiangolo.com](https://fastapi.tiangolo.com)

FasterAPI is a **separate** project with different internals (e.g. **msgspec**, radix routing); it is **not** affiliated with or endorsed by the FastAPI maintainers.

## Other building blocks

FasterAPI builds on **[msgspec](https://jcristharif.com/msgspec/)**, **[uvicorn](https://www.uvicorn.org/)**, and the **ASGI** spec, among others — see the [README](https://github.com/FasterApiWeb/FasterAPI#installation) for the full dependency picture.
